路由器环境搭建

首先是安装binwalk，有两种方法

方法1:sudo apt-get install binwalk 
方法2:
git clone https://github.com/attify/firmware-analysis-toolkit.git
这个工具包里有binwalk和firmadyne和qemu三个工具，不过qemu建议自己编译安装，所以可以把setup.sh里面安装qemu的部分注释掉
cd firmware-analysis-toolkit
sudo ./setup.sh
然后只需要等待即可，一键安装
安装完binwalk后提取squashfs-root文件系统有时会出问题，需要安装sasquatch
git clone https://github.com/devttys0/sasquatch.git
cd sasquatch
./build.sh

至此，binwalk安装完毕

接下来是qemu，按照H4lo大佬的说法，过高版本或者过低版本在某些情况下会报错，如果是Ubuntu16，推荐qemu2.4.0版本，如果是Ubuntu18推荐qemu2.10.0下载压缩包本地编译

wget https://download.qemu.org/qemu-2.4.0.tar.xz
tar xvf qemu-2.4.0.tar.xz && cd qemu-2.4.0/
./configure
make -j8
sudo make install

还安装一个qemu-user-static，因为qemu运行需要连接库，静态编译的qemu不需要库文件

apt install qemu-user-static

交叉编译环境 buildroot安装

还是跟着h牛的文章来，先把所有库都装上

sudo apt-get install libncurs*
wget http://buildroot.uclibc.org/downloads/snapshots/buildroot-snapshot.tar.bz2
tar -jxvf buildroot-snapshot.tar.bz2
cd buildroot/
make clean
make menuconfig
sudo make

make menuconfig之后，会出现一个GUI界面，进入Target option，选择Target Architecture Varient为Mips 32，将Target Architecture选为MIPS(little endian),保存后退出到上一级，进入Toolchain，将Kernel Headers修改为本机的kernel版本即可，然后保存退出，编译时间会有一个小时左右

gdb-multiarch

sudo apt-get install gdb-multiarch

接下来我们还需要安装一下mips版本的gcc及其相关组件，apt 中对应的包名为 gcc-mips-linux-gnu 与 gcc-mipsel-linux-gnu

sudo apt-get install gcc-mips-linux-gnu
sudo apt-get install gcc-mipsel-linux-gnu

相关的libc库都放在/usr目录下

其中mips-linux-gnu-gcc用来编译大端序程序，mipsel-linux-gnu-gcc用来编译小端序

我们来写一个小程序验证一下

#include <stdlib.h>
#include <malloc.h>


int main(){

        int *a;
        a = malloc(0x10);

        puts("demo");
        return 0;
}

这是h牛的文章中的一个程序

首先我们用小端序编译

mipsel-linux-gnu-gcc demo.c -o demo -static

静态编译

可以看出确实是小端序静态链接，接着我们运行一下

运行没问题。我们再用大端序编译一下

mips-linux-gnu-gcc demo.c -o demo -static

确实是大端序静态链接

接着运行

无误

现在都是静态链接，我们再动态编译一下

mipsel-linux-gnu-gcc demo.c -o demo

qemu-mipsel -L /usr/mipsel-linux-gnu/ ./demo

-L是连接指定的libc库

如果还需要安装arm的环境，可以运行如下命令

apt install gcc-arm-linux-gnueabihf

这个工具用来编译arm程序

然后用

apt-cache search "libc6" | grep arm

找到arm的库

将这些库安装好即可
