    打开环境，登录框

    随便登陆测试一下

    不出意外的失败

    登陆框的sql语句大概就是
    select * from table where username='$username' and password='$password'
    那么万能密码先走一波

    构造成
    select * from table where username='admin' or 1=1#' and password='123'
